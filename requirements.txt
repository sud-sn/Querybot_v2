# ⚠️  Azure SQL — OS-level driver REQUIRED in addition to pyodbc
# ---------------------------------------------------------------
# pyodbc is just the Python binding. The actual ODBC driver is an OS package
# and must be installed separately. Our code hard-codes
#   DRIVER={ODBC Driver 18 for SQL Server}
# in core/schema.py and core/examples.py, so version 18 specifically is needed.
#
#   Ubuntu / Debian:
#     bash install_linux_odbc18.sh
#
#   Or install it manually:
#     curl https://packages.microsoft.com/keys/microsoft.asc | sudo apt-key add -
#     curl https://packages.microsoft.com/config/ubuntu/$(lsb_release -rs)/prod.list \
#       | sudo tee /etc/apt/sources.list.d/mssql-release.list
#     sudo apt-get update
#     sudo ACCEPT_EULA=Y apt-get install -y msodbcsql18
#
#   macOS:
#     brew tap microsoft/mssql-release https://github.com/Microsoft/homebrew-mssql-release
#     brew update
#     HOMEBREW_ACCEPT_EULA=Y brew install msodbcsql18
#
#   Azure VM deploy.sh already installs msodbcsql18 automatically.
